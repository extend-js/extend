## 介绍

我们希望能在项目开发过程中更方便、省时，所以在原生js的基础上扩展了部分功能。希望对你在项目中有所帮助。

此存储库采用 [monorepo](https://en.wikipedia.org/wiki/Monorepo) 管理方式，使用 [yarn](https://yarn.bootcss.com/) 管理依赖.

首先，请安装 `yarn`:

```console
npm install yarn -g
```
